ChatGPT API Gateway

安全接入 ChatGPT API

前端请求本站接口，后端再安全调用 OpenAI API。API Key 放在服务器环境变量中，不暴露给用户浏览器。

How to use

部署步骤

1

安装依赖：npm install

2

复制：.env.example 改名为 .env

3

在 .env 填入新的 OPENAI_API_KEY

4

启动：npm start

API Docs

接口

GET /v1/status 查看服务状态

POST /v1/chat 简单对话接口

POST /v1/chat/completions 兼容 Chat Completions

GET /v1/models 查看模型列表

前端调用示例
fetch("/v1/chat", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    message: "你好"
  })
})
.then(res => res.json())
.then(console.log);

Security

安全提醒

不要把真实 API Key 写在 HTML、JS、GitHub 公开仓库里。正式部署时请使用服务器环境变量， 并增加访问鉴权、限流和日志记录。
